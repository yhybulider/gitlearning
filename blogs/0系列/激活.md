# Intellij Idea2019版本激活，目测可以激活到2089！！

## 前言

最近的idea都突然过期，即之前的激活码的都失败了，自己也试了两次自己激活码注册，导致时不时又失效了，今天查到了一些办法，总结出来给需要的人。这个方法可以激活到2089年。用了半个小时泄露这篇文章，想给大伙一点参考。都是借鉴来的，有不对的地方，多多指教！

<img src="https://i.loli.net/2020/01/10/2zJ3OW1ZYoS7csE.png" width="850px"/>

## 步骤（直接激活新下载的）

1. 先是链接: https://pan.baidu.com/s/1709ltiM5fIxZFYMjnROBKQ 提取码: r9dt

2. 里面有激活码破解码，破解补丁，需要两个配合使用的

   <img src="https://i.loli.net/2020/01/10/8BpNkngzWEi4r5J.png" width="850">

   <a href="https://sm.ms/image/XNL1kfCxJKgAzHs" target="_blank"><img src="https://i.loli.net/2020/01/10/XNL1kfCxJKgAzHs.png" width="850px"></a>

   

3. 将里面的.jar文件放置到你电脑装idea的路径里面的bin目录下

   <a href="https://sm.ms/image/AZ4CefIVXBGF8OQ" target="_blank"><img src="https://i.loli.net/2020/01/10/AZ4CefIVXBGF8OQ.png" width="850"></a>

   <a href="https://sm.ms/image/Q3FhlRq2e9rJ8n5" target="_blank"><img src="https://i.loli.net/2020/01/10/Q3FhlRq2e9rJ8n5.png" width="850px" ></a>

   

4. [下载链接](https://www.jetbrains.com/idea/)——idea下载，选择你想要的打开idea，

5. 新装的或者你重新装的话，可以免费试用30天进去，在help栏下点击-edit custom VM options

   

   <a href="https://sm.ms/image/GcO2KQVrUl4dxEy" target="_blank"><img src="https://i.loli.net/2020/01/10/GcO2KQVrUl4dxEy.png" width="850px" height="850"/></a>

6. 在最后一行加多一行 **-javaagent:你的安装路径\jetbrains-agent.jar**（就是idea的安装路径）

<a href="https://sm.ms/image/68Nr3MIwCxSBbAJ" target="_blank"><img src="https://i.loli.net/2020/01/10/68Nr3MIwCxSBbAJ.png" width="850px"></a>

## 成功

然后将重新打开idea，点击**`Activation code`方式激活**，选择一个激活码填进去，激活即可发现ok就可以点进去成功了

## 注意（不想重新下载的小伙伴）

有些人已经是失效了，已经进不去idea，也不想重新下载，那就先随便找个激活码，进去idea先，你才可以修改文件配置

[这里给出一个参考的链接](http://idea.medeming.com/jet/)——这个链接可以生成下载激活码，你也可以直接用但是这种感觉随时失效的，还是进去之后按上面的步骤下来做。
