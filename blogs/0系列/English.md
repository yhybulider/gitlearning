



# 英语积累

## 前言

这一篇blog是记录自己在学习过程中遇到的一些英文单词，特别是频率较高的。

- 在linux上的

| 单词     | 翻译                                  |
| -------- | ------------------------------------- |
| synopsis | 概要、大纲                            |
| mount    | 直译是组织安排，用来挂载Linux外的文件 |

- Java上的

| 单词       | 翻译           |
| ---------- | -------------- |
| subsequent | 参数           |
| allocate   | 分配           |
| surrogate  | 替代的         |
| radix      | 基数、根值     |
| pattern    | 模式，方法     |
| cache      | 隐藏物，储存物 |
| delimiter  | 定界符         |
| invocation | 调用           |
| semicolons | 分号           |
